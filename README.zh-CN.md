# 适用于 macOS 的 GenAI 提示词文本替换

[English](README.md)

![适用于 macOS 的 GenAI 提示词文本替换](assets/feature-image.png)

本仓库提供一套用于 GenAI 提示词工程的 macOS 文本替换词库，主要面向从事 AI/ML 工程、敏捷交付和 DevOps 工作的技术人员。

## 概览

仓库中的 `Text Substitutions.plist` 包含 50 条适用于 ChatGPT、Claude 和 Copilot 等 GenAI 工具的提示词快捷命令。这些命令覆盖专业沟通、敏捷工作流、DevOps 和技术领导力等常见场景。

### 版本 2.0 迁移说明

版本 2.0 统一使用以 `zz` 开头、便于记忆的快捷命令。常用操作采用熟悉的英文单词，例如 `zzemail`、`zzreply` 和 `zzplan`。几个容易混淆的评审命令使用不同名称：`zzpaper` 用于学术论文审稿，`zzcode` 用于机器学习代码审查，`zztech` 用于生产就绪技术评审。

如果你导入过旧版本，请先导出备份并删除旧条目，再导入新的 plist 文件。macOS 不会自动删除之前导入的快捷命令。

## 在 macOS 中导入文本替换

> **重要提醒：** 导入 plist 文件会向现有的 macOS 文本替换列表添加条目。这不是全量替换或重置操作。Apple 没有说明导入过程如何处理已经存在的快捷命令，因此你可能需要手动清理重复项或冲突项。导入前，请先导出当前文本替换列表作为备份。导入后，请检查列表，再开始使用新的快捷命令。

按照 Apple 支持的拖放方式导入：

1. 在 Mac 上打开 **系统设置**。
2. 在侧边栏中选择 **键盘**。
3. 点击 **文本替换**。
4. 在 Finder 中找到 `Text Substitutions.plist`。
5. 将 plist 文件拖入 **文本替换** 窗口。
6. 点击 **完成**。

如需导出备份，请在 **文本替换** 窗口中选择现有条目，然后将条目拖到 Finder。macOS 会创建一个 plist 备份文件，你可以在需要时重新导入。

## 验证步骤

导入后，请确认安装是否成功：

1. **检查系统设置**
   - 前往 **键盘** → **文本替换**
   - 确认列表中包含全部 50 条快捷命令
   - 确认每条记录都显示快捷命令（例如 `zzed`）和对应的提示词

2. **测试快捷命令**
   - 打开支持文本替换的应用，例如“备忘录”或 TextEdit
   - 输入一条快捷命令，例如 `zzed`
   - 按空格键或 Tab 键
   - 确认应用展开完整提示词

并非所有应用都支持文本替换。如果快捷命令可以在“备忘录”或 TextEdit 中使用，但无法在其他应用中生效，请检查对应应用是否支持 macOS 文本替换。

## 使用建议

### 最佳实践

- **统一前缀**：所有快捷命令都使用 `zz` 前缀，避免与常用单词冲突
- **易于记忆**：快捷命令尽量采用直观、易记的名称
- **切换上下文**：针对不同任务使用不同命令，帮助 AI 理解上下文
- **匹配语言**：除非你指定其他语言，提示词会使用与你输入相同的语言回复。`zztranslate` 使用你指定的目标语言
- **按需定制**：根据自己的工作方式自由修改提示词

### 示例工作流

1. **规划阶段**：使用 `zzplan`、`zzstory`、`zzsprint`
2. **开发阶段**：使用 `zzarch`、`zzpipeline`、`zzcode`
3. **部署阶段**：使用 `zzcicd`、`zzmlops`、`zztest`
4. **沟通阶段**：使用 `zzemail`、`zzstake`、`zznotes`
5. **文档阶段**：使用 `zzapi`、`zztutorial`、`zzreport`

### 自定义

如需添加或修改快捷命令：

1. 直接编辑 `Text Substitutions.plist`
2. 新增条目时沿用相同的 XML 结构
3. 使用上文的拖放方式重新导入 plist 文件
4. 重启应用以查看更新后的效果

## 常见问题

- **快捷命令无法使用**：先在“备忘录”或 TextEdit 中测试，再根据需要重启对应应用
- **出现重复项**：导入新版本前先删除旧条目
- **出现冲突项**：检查是否已经存在相同的快捷命令
- **同步问题**：如果需要在 Apple 设备之间同步，请启用 iCloud Drive，并在所有设备上登录相同的 Apple 账户

## 技术说明

- **文件格式**：Apple Property List（plist）XML 格式
- **导入方式**：将 plist 文件拖入 **系统设置** → **键盘** → **文本替换**
- **同步方式**：使用同一 Apple 账户时，通过 iCloud Drive 同步
- **备份方式**：在 **文本替换** 窗口中选择条目，并将条目拖到 Finder
